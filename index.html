Frontend

Definición

Tecnologías del lado del cliente

Las tecnologías frontend se centran en la interfaz de usuario y la experiencia del usuario en una aplicación web. Algunas de las tecnologías más comunes utilizadas en el desarrollo frontend son: HTML (HyperText Markup Language): Es el lenguaje de marcado estándar utilizado para estructurar el contenido de una página web. CSS (Cascading Style Sheets): Se utiliza para definir la apariencia y el diseño visual de una aplicación web. JavaScript: Es un lenguaje de programación que permite agregar interactividad y funcionalidad dinámica a una aplicación web. También es ampliamente utilizado en el desarrollo de frameworks y bibliotecas frontend. Además de estas tecnologías básicas, existen numerosos frameworks y bibliotecas frontend populares, como Angular, React y Vue.js, que proporcionan herramientas adicionales para el desarrollo web, facilitando la creación de interfaces de usuario complejas y reactivas.

HTML5

Es la última versión del lenguaje de marcado estándar para la creación y estructuración de contenido en la web. HTML (HyperText Markup Language) es el lenguaje base utilizado para describir la estructura y el contenido de una página web, y HTML5 es la versión más reciente y avanzada de este lenguaje.

CSS

Shout! Media Productions

(Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir y controlar la presentación y el diseño de un documento HTML o XML. Mientras que HTML se encarga de definir la estructura y el contenido de una página web, CSS se utiliza para definir cómo se ve y se presenta ese contenido.

Botstrap

Shout! Media Productions

Es un framework front-end de código abierto ampliamente utilizado para el desarrollo rápido y la creación de interfaces de usuario responsivas y estéticamente atractivas. Fue desarrollado por Twitter y ahora es mantenido por la comunidad de desarrolladores. Bootstrap se basa en HTML, CSS y JavaScript, y proporciona un conjunto de estilos predefinidos, componentes y herramientas que permiten agilizar el proceso de diseño y desarrollo de sitios web y aplicaciones web. Está diseñado para ser fácil de usar, adaptable y compatible con la mayoría de los navegadores web modernos

Backend

Definición

Lógica y funcionamiento
Las tecnologías backend están relacionadas con la lógica y el funcionamiento detrás de escena de una aplicación web. Algunas de las tecnologías comunes utilizadas en el desarrollo backend son: Lenguajes de programación: Los lenguajes populares para el desarrollo backend son Python, Ruby, Java, PHP, C#, Python, Go, entre otros. Estos lenguajes permiten la implementación de la lógica empresarial y el manejo de la comunicación con bases de datos y otros servicios. Frameworks backend: Hay una amplia gama de frameworks disponibles para diferentes lenguajes de programación. Algunos ejemplos populares son Django y Flask para Python, Ruby on Rails para Ruby, Spring para Java, Laravel, Slim para PHP, y ASP.NET para C#, Gorilla MUX para Go. Bases de datos: Para almacenar y gestionar datos, se utilizan sistemas de gestión de bases de datos como MySQL, PostgreSQL, MongoDB o SQLite. Estas bases de datos se integran con el backend para proporcionar almacenamiento persistente de datos. La elección de las tecnologías frontend y backend dependerá de los requisitos específicos de tu aplicación web y de tus preferencias personales o las de tu equipo de desarrollo.
